# 安全政策

## 支持的版本

我们只为最新版本的 SunloginManager 提供安全更新。

## 报告漏洞

如果您发现安全漏洞，请不要在公共 Issue 中报告，而是直接通过以下方式联系项目维护者：

- 发送邮件至：[您的邮箱地址]
- 通过 GitHub 私信联系：[@songtay](https://github.com/songtay)

请提供尽可能详细的信息，包括：

- 漏洞的详细描述
- 重现步骤
- 潜在影响
- 如果可能，提供概念验证代码

## 响应时间

我们会在收到漏洞报告后的 48 小时内确认收到，并尽快评估漏洞的严重性和影响范围。

## 修复流程

1. 确认漏洞并评估严重性
2. 开发修复补丁
3. 在发布安全更新前，会给报告者留出合理的时间（通常为 7-14 天）
4. 发布安全更新和公告

## 致谢

我们感谢所有负责任地报告安全漏洞的研究人员和用户。在发布安全更新后，我们会在致谢部分感谢贡献者（除非您希望保持匿名）。
