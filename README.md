# 向日葵远程连接管理器 (SunloginManager)

![GitHub release (latest by date)](https://img.shields.io/github/v/release/960427357/SunloginManager)
![GitHub license](https://img.shields.io/github/license/960427357/SunloginManager)
![.NET](https://img.shields.io/badge/.NET-8.0-purple.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

一个功能强大、界面美观的向日葵远程连接管理工具，采用现代化的 Apple 设计风格，提供安全的连接管理、分组功能和加密存储。支持一键快速连接，自动输入识别码和连接码，让远程连接管理更加高效便捷。

## ✨ 核心特性

### 🎨 现代化界面
- **Apple 风格设计**：圆角卡片、柔和阴影、流畅动画
- **直观的用户体验**：清晰的布局、一致的交互
- **响应式设计**：悬停效果、选中状态、视觉反馈

### 🔐 安全保护
- **连接码加密存储**：使用 Windows DPAPI 加密敏感信息
- **密码输入保护**：PasswordBox 隐藏输入内容
- **列表掩码显示**：连接码显示为 `********`
- **临时查看功能**：眼睛按钮按住显示，松开隐藏
- **用户级加密**：每个用户的加密密钥不同

### 📁 分组管理
- **连接分组**：按项目、客户或用途分类管理
- **颜色标识**：为每个分组设置不同颜色
- **分组筛选**：快速切换查看不同分组
- **批量管理**：支持分组的增删改查
- **默认分组**：系统自动创建，不可删除

### 🚀 快速连接
- **一键连接**：选择连接后点击按钮即可
- **自动填充**：自动输入识别码和连接码
- **键盘模拟**：使用 Tab 键切换输入框，模拟真实用户操作
- **智能等待**：自动检测窗口状态，适配向日葵 WebView 界面
- **多种连接方式**：列表连接、详情面板连接、表格内连接

### 📝 日志记录
- **详细日志**：记录所有操作和错误
- **日志查看器**：内置日志文件夹快速访问
- **按日期分类**：每天生成独立日志文件
- **问题排查**：便于定位和解决问题
- **日志级别**：支持 Info、Warning、Error 等级别

### 🔍 搜索过滤
- **实时搜索**：输入即时过滤连接列表
- **多字段匹配**：支持按名称、识别码搜索
- **分组过滤**：按分组快速筛选连接
- **组合过滤**：搜索和分组过滤可同时使用

### ⚙️ 灵活配置
- **自定义路径**：配置向日葵客户端路径
- **自动检测**：自动搜索常见安装路径
- **数据持久化**：JSON 格式存储，易于备份
- **自动迁移**：旧数据自动升级为新格式
- **系统托盘**：最小化到系统托盘，后台运行

### ⭐ v2.0 新增功能
- **双击连接**：双击列表项直接连接
- **收藏功能**：星标标记常用连接

- **连接统计**：成功率、最常使用 TOP5、7天/30天统计
- **连接测试**：验证识别码格式和客户端状态

## 📖 使用指南

### 添加连接

1. 点击主界面的"添加连接"按钮
2. 填写以下信息：
   - **连接名称**：便于识别的名称
   - **识别码**：向日葵设备识别码
   - **连接码**：远程连接密码（加密存储）
   - **所属分组**：选择或创建分组
   - **备注**：可选的说明信息
3. 点击"添加"保存

### 管理分组

1. 点击"管理分组"按钮
2. 可以进行以下操作：
   - **添加分组**：创建新分组，设置名称、描述和颜色
   - **编辑分组**：修改分组信息
   - **删除分组**：删除不需要的分组（连接会移到默认分组）
3. 分组会在主界面的筛选下拉框中显示

### 连接远程设备

1. 在列表中选择要连接的设备
2. 点击"快速连接"按钮（或表格中的连接按钮）
3. 应用会自动：
   - 启动向日葵客户端
   - 等待窗口加载完成
   - 输入识别码
   - 按 Tab 键切换到连接码输入框
   - 输入连接码（自动解密）
   - 按回车键建立连接
   - 更新最后连接时间

### 搜索连接

1. 在主界面顶部的搜索框中输入关键词
2. 支持搜索：
   - 连接名称
   - 识别码
3. 搜索结果实时更新
4. 可与分组过滤同时使用

### 查看连接码

**在列表中**：
- 连接码显示为 `********`，保护隐私

**在编辑对话框中**：
1. 打开"编辑连接"对话框
2. 找到连接码输入框
3. 按住右侧的眼睛图标按钮
4. 连接码以明文显示
5. 松开按钮，恢复为圆点显示

### 查看日志

1. 点击"查看日志"按钮
2. 自动打开日志文件夹
3. 日志文件命名格式：`app_YYYYMMDD.log`
4. 日志包含：
   - 操作记录（Info）
   - 警告信息（Warning）
   - 错误信息（Error）
   - 异常堆栈跟踪

### 系统托盘

1. 点击窗口关闭按钮，应用最小化到系统托盘
2. 双击托盘图标恢复主窗口
3. 右键托盘图标显示菜单：
   - 显示主窗口
   - 退出应用（需确认）

### 核心服务

#### EncryptionService（加密服务）
- 使用 Windows DPAPI 加密连接码
- 用户级加密（DataProtectionScope.CurrentUser）
- 每个用户密钥不同，无法跨用户解密
- 自动加密/解密，对用户透明
- 支持旧数据自动迁移
- 提供加密状态检测功能

#### DataService（数据服务）
- JSON 格式存储数据
- 支持连接和分组的 CRUD 操作
- 自动修复数据问题（如 ID 冲突）
- 数据迁移和升级（加密旧连接码）
- 初始化默认分组
- 数据完整性检查

#### SunloginService（向日葵服务）
- 启动向日葵客户端
- 自动检测常见安装路径
- 自动输入识别码和连接码
- 窗口检测和管理（适配 WebView）
- 键盘事件模拟（Tab、Enter、Ctrl+A）
- 智能等待和重试机制
- 支持验证码输入（可选）

#### LogService（日志服务）
- 按日期分类存储（app_YYYYMMDD.log）
- 支持不同日志级别（Info、Warning、Error）
- 异常详细记录（包含堆栈跟踪）
- 线程安全的文件写入
- 便于问题排查和调试

## 🔒 安全特性

### 加密存储
- **算法**：Windows DPAPI (Data Protection API)
- **范围**：CurrentUser（用户级）
- **熵值**：自定义熵值增强安全性
- **格式**：Base64 编码
- **特点**：
  - 每个用户的加密密钥不同
  - 无法跨机器或跨用户解密
  - 系统级安全保护
  - 自动检测加密状态
  - 兼容旧版本明文数据

### 数据格式
```json
{
  "id": 1,
  "name": "测试连接",
  "identificationCode": "123456789",
  "encryptedConnectionCode": "AQAAANCMnd8BFdERjHoAwE...",
  "groupId": 1,
  "remarks": "测试备注"
}
```

### 显示保护
- **列表显示**：`********`（固定8个星号）
- **输入显示**：`●●●●●●●●`（PasswordBox 圆点）
- **临时查看**：按住眼睛按钮显示明文
- **自动隐藏**：松开或移开鼠标自动隐藏

## 📂 数据存储

### 存储位置
```
%AppData%\SunloginManager\
├── Data\
│   ├── connections.json    # 连接数据（加密）
│   ├── groups.json          # 分组数据
│   └── settings.json        # 应用设置
└── Logs\
    ├── app_20250101.log     # 日志文件
    ├── app_20250102.log
    └── ...
```

### 备份建议
- 定期备份 `Data` 文件夹
- 加密数据只能在同一用户下恢复
- 跨机器迁移需要重新输入连接码
- 建议记录重要连接的明文信息

## 🔧 常见问题

### 无法找到向日葵客户端？
1. 打开设置对话框
2. 手动选择向日葵客户端的安装路径
3. 通常位于：`C:\Program Files\Oray\SunLogin\SunloginClient.exe`

### 连接失败怎么办？
1. 检查向日葵客户端是否正常运行
2. 确认识别码和连接码是否正确
3. 查看日志文件了解详细错误信息
4. 确保向日葵窗口没有被其他窗口遮挡

### 加密的连接码可以恢复吗？
- 加密数据只能在同一 Windows 用户账户下解密
- 更换电脑或用户账户后需要重新输入连接码
- 建议在安全的地方备份重要连接的明文信息

### 如何备份数据？
1. 复制 `%AppData%\SunloginManager\Data` 文件夹
2. 注意：加密数据只能在同一用户下恢复
3. 跨机器迁移需要重新输入连接码
